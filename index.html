Road Assistant Application

Road Assistant Mobile App is an app that give the driver some information about the road and his car like (Bumps, Traffic Light, Cross Roads, etc.…) helped him avoid the road obstacles.

First of all the application ask to allow the location of the user and then the user should connect the OBD to the car and connect it with the bluetooth from the side menu in Road Assistant Application. Handle the setting of the alarm and then you have an amaizing drive assistant.

The App screens Consist of:
  • Home screen that shows the car data like speed and can open the alarm on it.
  • Map screen that shows the user location and the data added before (Bumps, Traffic Lights, etc.…).
  • Add New Data screen allow the user to add new road data after register or login.
  • Setting screen allow the user to change the alarm setting.
  • Account screen if the user login shows its data otherwise show the login and sign-up screen.
